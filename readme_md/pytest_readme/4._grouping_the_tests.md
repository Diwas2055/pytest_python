# Grouping the Tests

we will learn how to group the tests using markers.

Pytest allows us to use markers on test functions. **Markers are used to set various features/attributes to test functions**. Pytest provides many inbuilt markers such as xfail, skip and parametrize. ***Apart from that, users can create their own marker names.*** Markers are applied on the tests using the syntax given below âˆ’

```python
@pytest.mark.<markername>

```

To use markers, we have toÂ **import pytest**Â module in the test file. We can define our own marker names to the tests and run the tests having those marker names.

To run the marked tests, we can use the following syntax âˆ’

```bash
pytest -m <markername> -v
```

<aside>
ðŸ’¡ ***m <markername>*** represents the marker name of the tests to be executed.

</aside>

Update our test filesÂ **test_compare.py**Â andÂ **test_square.py**Â with the following code. We are defining 3 markersÂ **â€“ great, square, others**.

## **test_compare.py**

```python
# **test_compare.py**

import pytest

@pytest.mark.great
def test_greater():
   num = 100
   assert num > 100

@pytest.mark.great
def test_greater_equal():
   num = 100
   assert num >= 100

@pytest.mark.others
def test_less():
   num = 100
   assert num < 200
```
